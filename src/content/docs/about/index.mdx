![Alec Estrada](/images/profile/profile.jpg)
I’m a software engineer with experience supporting distributed, cloud-native services in production environments.

At Optum / WellMed, I operate and troubleshoot real-time streaming systems deployed across Azure and hybrid Linux-based infrastructure. My responsibilities include production incident response, log and metric analysis, Kubernetes troubleshooting, and cross-system remediation during high-severity outages.

I design and maintain containerized Python microservices orchestrated through Kubernetes clusters and Kafka-based streaming platforms. I also build and maintain CI/CD pipelines, manage secure cloud identities, and implement monitoring through OpenTelemetry and Grafana dashboards.

Recently, I’ve been expanding into AWS infrastructure by building event-driven systems using Lambda, SQS, ECS Fargate, and IAM role separation. These projects focus on durability, failure isolation, and operational clarity.
