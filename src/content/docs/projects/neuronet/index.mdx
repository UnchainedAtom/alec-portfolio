## Overview

Neuronet is a web-based platform used to experiment with dynamic system behavior, user input handling, and backend processing workflows.

It functions as a sandbox for exploring how state changes propagate through frontend and backend components, including input validation and file handling.

Simulates artificial NFT inspired economy, with image upload features.

Originally built as a personal project for a hombrew DnD Sci-fi campaign, it’s now focused on being a general platform for testing interactive system patterns.
