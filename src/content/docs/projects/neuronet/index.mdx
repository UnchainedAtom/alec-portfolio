## Overview

Neuronet is a personal platform project that explores dynamic systems and interactive data workflows in a web context. It’s built to be both a visual and functional sandbox for experimenting with stateful interactions, user inputs, and backend processing.  Orginally created for a hombrew Sci-fi DnD game.

Features include:
- Dynamic simulation state updates
- User-driven parameter configuration
- Image upload handling and validation

While in active development, the project demonstrates robust input processing, UI-to-backend communication, and iterative improvement of code quality and system feedback loops.

## Tech Stack
- Frontend (HTML/CSS/JS)
- Backend (Python/Flask)
- File handling and economy simulation logic

## Highlights
- Explores interactive system design
- Handles user uploads with validation
- Demonstrates integration of frontend and backend workflows
