## Overview

Neurox Terminal is a command driven interface built as a lightweight home system console. The focus is on rapid, text-based interactions with backend functionality.

Users can enter commands, trigger backend workflows, and see immediate feedback, making it useful for system exploration and early prototyping of operational interfaces.

## Tech Stack
- Frontend (HTML5, Vanilla JavaScript)
- Terminal style UI components (CSS3 (custom terminal styling))
- Integration with backend APIs

## Highlights
- Designed UI/UX for high-throughput command interaction
- Useful for rapid iteration and testing of backend services
- Demonstrates systems integration and frontend/backend communication
